# :computer: Technology Stack
- NodeJS
- TypeScript
- Express
- PostgreSQL
- TypeORM
- uuidv4
- ts-node-dev
- ESLint + Prettier

# :barber: How to use
```shell
  # Clone this repository
  - git clone https://github.com/izabela-am/GoBarber-API.git
  
  # Enter project directory
  - cd GoBarber-API
  
  # Install dependencies
  - yarn
  
  # Start the server
  - yarn dev:server
```
